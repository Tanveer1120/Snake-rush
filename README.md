🐍 Ultra Realistic Funny Snake Game
A fun and visually enhanced version of the classic Snake game, built using HTML, CSS, and JavaScript. This game combines smooth animations, glowing effects, and a humorous touch to create a unique gameplay experience.
🎮 Features
🐍 Smooth and glowing snake design
😂 Funny snake face with animated eyes
🍎 Shiny animated food
📈 Real-time score tracking
💥 Game over alert with humor
⚡ Lightweight and fast (runs in browser)
🆓 No installation required
🚀 How to Run
Download or clone this repository
Locate the file:
index.html
Double-click the file OR open it in any browser
👉 That’s it! The game will start instantly.
🕹️ Controls
Use your keyboard arrow keys:
⬅️ Left Arrow → Move Left
➡️ Right Arrow → Move Right
⬆️ Up Arrow → Move Up
⬇️ Down Arrow → Move Down
